Proyectos

  • Sistema informático para una empresa de venta de componentes

    Este proyecto se basa en una serie de mini proyectos que, al combinarse, permitirán a nuestro grupo crear una empresa determinada.

    En el primer mini proyecto, en el área de sistemas, se nos solicitó desarrollar un servicio DHCP con tres tarjetas de red, cada una asignando diferentes rangos de direcciones IP. Además, implementamos un servicio DNS con registros directos e inversos.

    En el segundo mini proyecto, implementamos un directorio activo (Active Directory), el cual crearemos cinco usuarios con sus respectivas carpetas personales utilizando la variable %USERNAME%. Utilizamos la máquina del primer mini proyecto para implementar el servicio FTP y permitir la coexistencia de dos sitios web que responden a diferentes URLs.

    En el tercer mini proyecto, implementamos el servicio Mysql server. Para ello, reemplazamos la página web anterior y agregamos una base de datos creada por otro miembro del grupo, la cual debía ser visible desde cualquier navegador web. Para completar el proyecto, regresamos al Active Directory y creamos tres departamentos distintos con sus respectivos usuarios, y configuramos una unidad de red, la cual se conecta automáticamente al iniciar sesión y que realice una copia de seguridad del usuario al cerrar sesión, utilizando un script de inicio de sesión.

  • Cableado estructurado para una empresa de Madrid

    Este proyecto fue el primero en el ambito informatico, debíamos realizar un cableado estructurado mediante unas canaletas con diferentes rosetas para conectar cables Ethernet, en este proyecto estuve en todo el proyecto, haciendo los agujeros para las rosetas, crimpar los cables para realizar la conexión y la documentación que debíamos entregar al final de este.
